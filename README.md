# Sudan Vehicle Finder

## Project Overview
Sudan Vehicle Finder is a web application designed to help users find vehicles available for sale across Sudan. The application provides a user-friendly interface where buyers can browse listings, filter search results based on various criteria, and contact sellers directly.

## Features
- **Search Functionality:** Users can search for vehicles using keywords, price ranges, and other filters.
- **User Accounts:** Buyers and sellers can create accounts to manage their listings and inquiries.
- **Real-time Notifications:** Users receive notifications about new listings and inquiries.
- **Multilingual Support:** The application supports both English and Arabic languages.

## Technologies Used
- **Frontend:** React.js, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/5Hakim5/sudan-vehicle-finder.git
   ```
2. Navigate to the project directory:
   ```bash
   cd sudan-vehicle-finder
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Navigate to the homepage to start searching for vehicles.
- Create an account to list your vehicle for sale or to save your favorite listings.
